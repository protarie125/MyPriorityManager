# MyPriorityManager

1. "Add task" 버튼(Ctrl+A)으로 Task를 추가할 수 있습니다.
2. "+/-" 버튼으로 Priority를 1씩 변경할 수 있습니다.
3. "^/=" 버튼으로 Priority를 상대적으로 변경할 수 있습니다.

File|New, Open, Save는 버튼만 있습니다 하하
